# Lab1
В нашем проекте представлены 4 функции по геометрическим фигурам:

1. Прямоугольник - находит периметр, площадь, длину диагонали по заданным сторонам.
2. Треугольник - находит периметр, площадь по формуле Герона, проверяет равнобедренность по заданным сторонам.
3. Трапеция - находит периметр, площадь, длину средней линии по заданным сторонам.
4. Окружность - находит длину окружности, площадь круга, площадь кругового сектора по заданному радиусу.
