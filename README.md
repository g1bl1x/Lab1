# Lab1
В нашем проекте представлены 4 функции по обработке строки:

1. Подсчет гласных букв - определяет количество гласных (a,  e,  i,  o,  u,  y,  а,  е,  ё,  и,  о,  у,  ы,  э,  ю,  я) в строке без учета регистра
2. Преобразование регистра - переводит все буквы строки в верхний или нижний регистр
3. Очистка от специальных символов - удаляет все знаки препинания и управляющие символы, оставляя только буквы, цифры и пробелы
4. Проверка на палиндром - определяет, читается ли строка одинаково слева направо и справа налево
